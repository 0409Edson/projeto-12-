Sobre este Exercício

Este projeto foi desenvolvido como parte do Módulo 12 do curso de Front-end da EBAC.

O objetivo principal foi praticar a utilização do framework Bootstrap para:

  • Criação de formulários modernos e responsivos.
  • Utilização de componentes como Navbar e Tabs.
  • Uso de utilitários de margem, espaçamento e sistema de grid.
  • Integração via CDN.
